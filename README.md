Linux Basics Assignment

This repository contains my submission for the Linux Basics Assignment.
It includes Linux commands, explanations, and screenshots demonstrating core file and system operations.

Topics Covered
1. Creating and Renaming Files / Directories

Creating directories using mkdir

Creating empty files using touch

Renaming files using mv

2. Viewing File Contents

Displaying full file content with cat

Viewing beginning of files using head

Viewing end of files using tail

3. Searching for Patterns

Using grep to search text inside files

4. Zipping and Unzipping Files

Compressing directories using zip

Extracting archives using unzip

5. Downloading Files

Downloading files from the internet using wget

6. Changing File Permissions

Modifying file permissions using chmod

7. Working with Environment Variables

Creating variables with export

Accessing variables using echo
